<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<p>Clone the repository-</p>
<pre><code>git clone git@github.com:kowsar52/To-do-List-Application----Laravel.git
</code></pre>
<p>Then cd into the folder with this command-</p>
<pre><code>cd Folder-name
</code></pre>
<p>Then do a composer install</p>
<pre><code>composer install
</code></pre>
<p>Then create a environment file using this command-</p>
<pre><code>cp .env.example .env
</code></pre>
<p>Then edit <code>.env</code> file with appropriate credential for your database server. Just edit these two parameter(<code>DB_USERNAME</code>, <code>DB_PASSWORD</code>).</p>
<p>Then create a database named <code>todos</code> and then do a database migration using this command-</p>
<pre><code>php artisan migrate
</code></pre>
<p>At last generate application key, which will be used for password hashing, session and cookie encryption etc.</p>
<pre><code>php artisan key:generate
</code></pre>

<p>Run server using this command-</p>
<pre><code>php artisan serve
</code></pre>
<p>Then go to <code>http://localhost:8000</code> from your browser and see the app.</p>
